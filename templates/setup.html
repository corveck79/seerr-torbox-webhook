Welcome.

Mycelium will turn your Seerr requests into Jellyfin-ready streams via TorBox in about 30 seconds. This wizard takes a few minutes; everything is hot-reloadable so nothing is final until you click Finish on the last step.

If you'd rather edit .env directly, click Skip wizard at the bottom and head to the dashboard.

TorBox

Your TorBox API key. Find it at torbox.app → Settings → API Keys. Without this Mycelium can't do anything.

Jellyfin

Optional but recommended. Mycelium triggers library scans after adding new content. Get the API key from Jellyfin → Dashboard → API Keys.

Seerr

Optional but recommended. Used to resolve IMDB IDs for requests that don't include one, and for catch-up sync of approved movies. From Jellyseerr/Overseerr → Settings → General → API Key.
