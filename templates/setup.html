When enabled, torrents only enter TorBox when you press play, and leave after idle time. Lets your library grow unlimited while staying within TorBox's 30-day cache window.

Trakt

Trakt scrobbles your watch progress from the web player — keeps your history, ratings and watchlist in sync. → trakt.tv/oauth/applications/new

After saving, connect your personal Trakt account via the profile menu in the SPA.

OpenSubtitles

Provides subtitle search in the web player. Free tier gives 500 downloads/day, VIP gives 1 000. → opensubtitles.com/consumers

Comma-separated ISO codes. Tried in order — nl,en prefers Dutch, falls back to English.

Zilean

Optional self-hosted torrent index (DMM-based). Faster than Torrentio for content in its index, no rate limits. Leave blank if you don't run one. → github.com/iPromKnight/zilean

Radarr & Sonarr

Optional bulk import of your existing Radarr/Sonarr libraries into Mycelium. API key: Radarr → Settings → General → Security — same path in Sonarr.
